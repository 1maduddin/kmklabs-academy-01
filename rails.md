## Rails Architecture ##

* MVC : Model View Controller
* ORM : Object Relational Mapping
* Helper Module : 

    * Email
    * Resource Management (js, css, img)

## Komponen MVC ##

* Model : ActiveRecord (ORM)
* View : Template Engine (Erb)
* Controller : ActionPack

    * ActionController
    * Action Dispatcher

## Komponen ORM ##

* ActiveRecord
* Database Migration
    
    * Schema : create, alter
    * Data : seed

## Testing ##

* Built-in

* Tambahan
    
    * rspec
    * capybara
